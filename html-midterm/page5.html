Changelog for HTML Midterm

I debugged the HTML Midterm using W3 Validator. Errors will be denoted below, followed by the solution.

Page 1

  1. Error: The character encoding was not declared
    Solution: Include UTF-8 character set in the head<meta charset="UTF-8">

  2. Error: Missing closing tr and table tags
    Solution: Include closing tr and table tags</tr> </table>

  3. Warning: Missing a language classification
    Solution: Include language classification in the html tag<html lang="en">

Page 2

  1. Error: Empty tag should be a closing bold tag
    Solution: Include closing bold tag </b>

  2. Error: Header element cannot be nested within a text-style element (in this case bold)
    Solution: Reorder element nesting <h2><b>Interior</b></h2>

  3. Error: Source referenced by an anchor should be linking to a reference on segmental arches, not "I don't know where to go"
    Solution: Replace source with an absolute link <a href="https://en.wikipedia.org/wiki/Segmental_arch"></a>

  4. Error: Missing closing header tag for H6; instead, includes opening hr tag
    Solution: Replace opening hr tag with closing h6 tag </h6>

Page 3

  1. Error: Empty head
    Solution: Include title, meta data, and link to associated stylesheet
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Midterm Page 3</title>
    <link rel="stylesheet" href="/html-midterm/css/cars.css">

  2. Error: Missing !DOCTYPE
    Solution: Add !DOCTYPE <!DOCTYPE html>

  3. Warning: Missing a language classification
    Solution: Include language classification in the html tag<html lang="en">

Page 4

  1. Error: Missing !DOCTYPE
    Solution: Add !DOCTYPE <!DOCTYPE html>

  2. Error: All images missing alternative text describing each image
    Solution: Add alternative text to each image; example <img src="testImage.jpg" alt="Example Descriptive Text">

  3. Error: Missing td element to wrap around image
    Solution: Add opening and closing td tags to nest around image <td></td>

  4. Error: Missing opening td tag
    Solution: Add opening td tag <td>
